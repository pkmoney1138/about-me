Projects

What follows are the projects I have created while taking Thinkful's Front End Development course. I believe that these projects are indicative of the knowledge I have accumulated while working through Thinkful's program.

New York Times Hack

This project was meant as an exercise in familiarizing ourselves with Google Chrome's Developer Tools. Using these developer tools, we had to change the front page of the New York Times in any way we saw fit. I decided to have a little fun with the headlines of that day.

Karma Landing Page Clone

Next, we were tasked with cloning the front page of another website.

Street Fighter

As an introduction to JavaScript and, more specifically, jQuery, we created this project. Positioning and hiding/showing elements were also a big part of this app.

Shopping List App

You don't have to be into "The Clash" or punk music in general to enjoy this shopping list app, but it helps!

Hot or Cold App

I didn't design this project, but I did get it to function properly with jQuery. This one's less about design and more about programming.

Superhero Quiz

A fun(I hope) interactive quiz about superheroes. This project was all about creating and using objects with JavaScript and jQuery.

API-Hack

Last, but not least, we learned to interact with APIs and how to extract data from them in object form and then use that data in some form on a website. I decided to interact with a movie database api and show poster images and some cast and crew info for any movie a user may want to look up.
